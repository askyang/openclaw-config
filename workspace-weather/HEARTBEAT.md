# HEARTBEAT.md - 天气助手定时任务

## 定时任务配置

每天8点（北京时间）执行天气播报任务：

```yaml
schedule: "0 8 * * *"  # 每天8:00执行
timezone: "Asia/Shanghai"
```

## 任务内容

1. 获取北京最新天气信息
2. 整理播报内容（包含温度、天气状况、风力、湿度、空气质量）
3. 添加生活建议（穿衣、出行建议）
4. 发送到指定群聊：oc_cf380451c6f74d1584a778544225c1ff

## 播报模板

```
🌤️ 北京天气播报 | {日期}

📍 地点：北京
🌡️ 温度：{最低温度}°C ~ {最高温度}°C
☁️ 天气：{天气状况}
💨 风力：{风力等级}，{风向}
💧 湿度：{湿度}%
🌫️ 空气质量：{空气质量指数} ({空气质量等级})

📝 生活建议：
{穿衣建议}
{出行建议}

祝您有美好的一天！🌤️
```

## 注意事项

- 使用 weather 技能获取准确天气数据
- 确保在8:00准时发送
- 如遇特殊天气（暴雨、大雪、雾霾等），添加特别提醒
- 保持温暖亲切的播报语气
